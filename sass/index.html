# 为什么需用学习Sass?

主要分为以下两点

  • 完美兼容CSS3，在CSS语言基础上添加了扩展功能，比如 变量、嵌套 (nesting)、混合 (mixin)。 而这些功能却正是我们工作所迫切需要的，且这些工作中常用功能在接下来的章节会分别有篇幅进行讲解。
  • 方便阅读同事或开源项目代码，在我们公司的项目中都统一使用Sass作为css的预处理语言，同时在大型有名的开源项目中，例如前端vue组件库 Element 的源码中就是使用的Sass

基于以上两点个人认为Sass还是有必要花时间精力一学的，废话不多少，马上来看看Sass提供了哪些好玩的东西吧。
