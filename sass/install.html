# 安装与编译

# 安装：

npm install -g sass

# 编译：

sass --watch input.scss output.css

  • input.scss为需要编译的Sass文件 （文件名称可根据实际需求自行更改）
  • output.css为编译后的css文件 （文件名称可根据实际需求自行更改）
